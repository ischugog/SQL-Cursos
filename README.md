Proyecto: Sistema de Cursos Escolares Lenguaje: 

SQLite (usando VS Code + extensión SQLite Viewer) Archivos:

- cursos.sql – crea la tabla CURSOS(id_curso, nombre, créditos, fecha_inicio).
- - estudiantes.sql – crea ESTUDIANTES(id_est, nombre, edad, correo).
- - maestros.sql – crea MAESTROS(id_mae, nombre).
- - inscripciones.sql – crea INSCRIPCIONES(id_est, id_curso,).
- - main.db – la base completa.
